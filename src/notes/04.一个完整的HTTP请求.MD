# 4. 一次浏览器完整的HTTP请求
    - 域名解析
    - 建立连接
    - 发起HTTP请求
    - 响应HTTP请求
    - 解析响应HTML
    - 加载HTML资源文件
    - 渲染呈现
    
### a.**域名解析**
- 查询浏览器自身DNS缓存`1分钟有效` `1000条缓存`
- 查询操作系统DNS缓存
- 查询hosts文件
- 向运营商DNS服务器发送请求
    - 运营商DNS服务器查询缓存
    - 迭代DNS请求，请求根域获得 -> com ，baidu.com -> www.baidu.com
- TCP三次握手建立连接
- 组织一个HTTP请求报文
- 服务器处理响应
    - 负载均衡
    - 查询缓存
    - 远程调用
    - 发送mq通知
    - 查询数据库（等等）
    - 得出结果
    - 组织HTTP响应报文
- TCP四次挥手释放连接
- 浏览器解析HTML代码
- HTML包含资源 先查缓存，再请求服务器
- 渲染
- 呈现
